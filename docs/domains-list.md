# domains-list

Lists all domains registered in the OpenPanel system.

## Synopsis

```
opencli domains-list [--user USERNAME] [--format FORMAT] [--help]
```

## Description

The `domains-list` command displays information about all domains registered in the OpenPanel system. You can filter by username and choose different output formats to suit your needs.

## Options

- `--user USERNAME`
  Filter domains to show only those belonging to the specified user.

- `--format FORMAT`
  Specify the output format. Available options:
  - `table` (default): Display results in a formatted table
  - `json`: Output in JSON format for programmatic processing
  - `csv`: Output in CSV format for importing into spreadsheets
  - `simple`: Output only domain names, one per line (useful for scripting)

- `--help`
  Display help information for this command.

## Examples

List all domains in the default table format:
```
opencli domains-list
```

List domains belonging to user 'john':
```
opencli domains-list --user john
```

List all domains in JSON format:
```
opencli domains-list --format json
```

List domains for user 'jane' in CSV format:
```
opencli domains-list --user jane --format csv
```

Get a simple list of domain names for scripting:
```
opencli domains-list --format simple
```
